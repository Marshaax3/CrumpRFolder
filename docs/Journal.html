Problem 5

Write a function to find the sum of all integers between any two values.

x<-c(1:100)

SumFun<-function(y){
  sum<-0
  for(i in 1:length(y)){
    sum=sum+y[i]
  }
print(sum)
}
SumFun(x)

    if(n==2){
      TRUE
